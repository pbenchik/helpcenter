# Update 27.09.2024

## Пересмотр системы добавления пользователей внутри компании

{% hint style="info" %}
**Было:** добавление пользователя в спейс и выдача прав на определенный спейс.
{% endhint %}

{% hint style="success" %}
**Стало:** добавление пользователя в компанию и распределение прав доступа между спейсами из участников компании.
{% endhint %}

## Добавлена роль Buyer & Buyer Team Lead

**Роль Buyer** - открывает доступ к трекеру, байер не видит никаких кампаний и статистики, только собственные. Каждому байеру можно настроить Тим Лида, который будет видеть всю его статистику.

**Роль Buyer Team Lead** - видит только свои кампании и байеров, которые находятся под его руководством. Одному байеру можно установить только одного лида.&#x20;



{% hint style="info" %}
в последующих обновления страница "Пользователи" будет доработана и там добавится отображение всех доступных спейсов возле каждого участника вашей компании с возможностью быстрого редактирования.
{% endhint %}
